Using helper classes & Sass mixins

Attach a helper class to the html element or use the mixin in your scss styles.

headline .headline
@include font-headline;
header .header
@include font-header;
subheader .subheader
@include font-subheader;
title .title
@include font-title;
subtitle .subtitle
@include font-subtitle;
body .body
@include font-body;
body-alt .body-alt
@include font-body-alt;
caption .caption
@include font-caption;
caption-alt .caption-alt
@include font-caption-alt;
